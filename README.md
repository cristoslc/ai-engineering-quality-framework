# AI Engineering Quality Framework

A comprehensive analysis of quality controls in the `~/.agents/AGENTS.md` hub-and-spoke framework, benchmarked against industry-standard QA/QC practices from software engineering (ISO 25010, IEEE 829, ISTQB, CI/CD quality gates) and manufacturing (Six Sigma, TQM, SPC, ISO 9001).

## Quick Start

The primary artifact is `docs/QUALITY-FRAMEWORK.md` — the full analysis with coverage mapping, gap analysis, and recommendations.

## Structure

- `docs/QUALITY-FRAMEWORK.md` — the Quality Framework document
- `docs/troves/` — research source materials
- `docs/domain-architecture/` — domain model of the quality framework itself
- `docs/technical-architecture/` — C4 diagrams of the quality control system
