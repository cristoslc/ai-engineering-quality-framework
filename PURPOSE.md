# Purpose

This project analyzes the quality controls embedded in an AI-assisted software engineering agent framework (`~/.agents/AGENTS.md`), benchmarking them against industry-standard QA/QC practices from software engineering (ISO 25010, IEEE 829, ISTQB) and manufacturing (Six Sigma, TQM, SPC, ISO 9001). It also compares the framework against Claydon's "Stop Reviewing Code. Start Reviewing Intent." thesis. The primary human-facing artifact is a report for framework owners; the full technical analysis is available for machine parsing. All content is AI-generated with no human review.
